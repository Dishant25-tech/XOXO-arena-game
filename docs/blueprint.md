# **App Name**: Tic Tac Toe Duel

## Core Features:

- Game Board: Display a 3x3 grid for the Tic Tac Toe game.
- Player Turn: Alternates turns between two players (X and O).
- Move Selection: Allows players to select an empty cell to place their mark.
- Win Condition: Checks for a win (3 in a row) or a draw after each move.
- Game Over: Announces the winner or a draw, and provides an option to start a new game.

## Style Guidelines:

- Primary color: Dark purple (#4A148C) to create a sense of playful competition.
- Background color: Very light desaturated purple (#F2E8FF).
- Accent color: Electric blue (#7DF9FF) for interactive elements, such as highlighting the winning row or the current player's turn.
- Body and headline font: 'Inter' for a clean, modern interface.
- Use simple and easily distinguishable 'X' and 'O' icons.
- Center the game board on the screen with sufficient padding.
- Use a subtle animation to indicate a player's turn.
